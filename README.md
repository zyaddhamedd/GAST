# GAST Industrial E-commerce

A high-end, premium industrial e-commerce website for the **GAST** brand, built with Next.js and Tailwind CSS. The project features a modern, responsive design optimized for both desktop and mobile devices.

## Features

- **Premium UI/UX**: Modern industrial aesthetic with smooth animations and transitions.
- **Mobile Optimized**: Custom mobile layouts for the homepage, store, product details, and support pages.
- **Dynamic Shop**: Advanced filtering by category, price, power, and voltage.
- **Interactive Product Details**: Image zoom, sticky call-to-action, and technical specification tabs.
- **Comprehensive Support**: FAQ and Contact pages with direct WhatsApp integration.

## Technologies Used

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: CSS Transitions & Framer Motion (where applicable)

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zyaddhamedd/GAST.git
   ```

2. Navigate to the project directory:
   ```bash
   cd gast-web
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

To start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To build the project for production:
```bash
npm run build
```

To start the production server:
```bash
npm run start
```
